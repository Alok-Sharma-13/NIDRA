NIDRA

Network Intrusion Detection
& Response Assistant

Real-time intrusion detection and response for modern web applications. NIDRA monitors your traffic and blocks threats before they reach your app.

NIDRA is an open source project. Anyone can use it, improve it, or add to it. If you have an idea or want to contribute, feel free to open a pull request on GitHub.

What it does

NIDRA is middleware you add to your app. It inspects every HTTP request and blocks attacks in real time.

🔍 Detection

Catches SQL injection, XSS, path traversal, command injection and more.

⚡ Blocking

Blocks malicious requests automatically and logs every event.

🌍 GeoIP

Resolves attacker IPs to countries and cities.

📊 Dashboard

Local web UI to see live attack logs and rule performance.

🧩 Easy setup

Works with Express, FastAPI, Flask, Django and more. ~5 min to integrate.

⚙️ Rule-based engine

9 OWASP-based detection rules covering the most critical web attack categories.

How it works

NIDRA sits between incoming requests and your application logic.

HTTP Request  →  NIDRA Middleware  →  Rule Engine  →  Your App
                       |
             GeoIP · Logger · Alerts · Dashboard

Quick start

  1. Run pip install nidra-sdk
  2. Copy nidra_sdk.py into your project directory
  3. Start the backend with python backend/app.py
  4. Apply the SDK decorator to your routes
That's it. NIDRA will now monitor and analyze every request against OWASP detection rules.

About NIDRA

A cybersecurity monitoring and response system built to detect malicious web activity and respond to threats in real time.

The Project

NIDRA demonstrates how rule-based intrusion detection systems can protect web applications from common attack vectors. It was built as an academic project exploring the practical design and implementation of a developer-friendly security layer.

The system intercepts incoming HTTP traffic, evaluates requests against a set of OWASP-based security rules, and responds in real time — blocking threats before they reach your application.
